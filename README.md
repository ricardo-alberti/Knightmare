# Knightmare Chess Engine

Open source chess engine

## UCI commands

### set up a position:
```bash
position startpos
position startpos moves e2e4 e7e5
```
### start search for best move:
```bash
go
```
### enable CLI debug UI:
```bash
debug on
```
